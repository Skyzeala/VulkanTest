environment setup 
please note this has only been tested on windows

Install 
    VS Code      https://code.visualstudio.com/
        Extensions
            C/C++
            C/C++ Extension Pack
            CMake Tools
            Shader languages support for VS Code
    Visual Studio (need the tools) https://visualstudio.microsoft.com/downloads/
    VulkanSDK   https://vulkan.lunarg.com/
    MSYS2       https://www.msys2.org/
        open msys2.exe and run: pacman -S --needed base-devel mingw-w64-x86_64-toolchain
        open mingw64.exe and run: pacman -S gcc make mingw-w64-x86_64-gcc mingw-w64-x86_64-mesa mingw-w64-x86_64-glew mingw-w64-x86_64-glfw
                                  pacman -S mingw-w64-x86_64-vulkan-devel
                                  pacman -S mingw-w64-x86_64-glm
                                  pacman -S mingw-w64-x86_64-cmake



Build steps:
    Open mingw64.exe and navigate to the build folder, then run these commands
    rm -rf CMakeCache.txt CMakeFiles VulkanTest.exe
    cmake .. -G "MinGW Makefiles" -DCMAKE_C_COMPILER=C:/msys64/mingw64/bin/gcc.exe -DCMAKE_CXX_COMPILER=C:/msys64/mingw64/bin/g++.exe
    mingw32-make

    You can compile the shaders to SPIR-V with the bat file in the shaders folder or with these commands in mingw (change path as needed)
    C:/VulkanSDK/1.4.313.0/Bin/glslc.exe fragment_shader.frag -o frag.spv
    C:/VulkanSDK/1.4.313.0/Bin/glslc.exe vertex_shader.vert -o vert.spv
